# Terminología en Español (es-419) — Canon v1

Status: ratified with Phase 3G, July 2026. One concept, one word —
identical across the interface, the catalog, and future marketing. This
document is a PRODUCT DECISION, the Spanish counterpart of
docs/constitution/terminology.md. Deviating copy is a bug. Amended,
never silently rewritten.

Register: neutral Latin American Spanish (es-419). Formal address
(usted), largely through impersonal constructions; controls use
infinitives ("Guardar", "Cancelar") per Spanish interface convention;
instructional prose stays calm and editorial. No Spain-specific
vocabulary (vosotros, ordenador, fichero) unless no broadly understood
Latin American equivalent exists.

## Grammar and typography rules

- Sentence-style capitalization; English title-case is not carried over
  except for ratified proper names (Copia de Lectura, Esquema Maestro,
  la Consigna as a label, lifecycle stage names as record facts).
- Status labels agree with their governing noun: versión → activa /
  reemplazada; hallazgo → abierto / resuelto / apartado; sesión →
  completa / incompleta / detenida; cuenta → vinculada.
- Prefer complete ICU messages over fragment assembly; plural messages
  carry their nouns and adjectives together ("# borradores abiertos").
- Spanish quotation marks («…» or “…”) in interface copy that quotes;
  stored content is never touched.
- Dates render through the locale seams (formatDate → "11 de julio de
  2026"); numbers group per es-419.
